# Line Notify Sample

Line Notifyを無理矢理まとめた

# Function
- sendMessage
- sendSticker
- sendImage
- sendImageWithURL

# Requirement

* Line-Notify-Token

LINE Notifyのトークン取得方法

[ここ](https://notify-bot.line.me/ja/)からアカウントを登録してトークンを取得

アカウント作成後、マイページ -> トークンを発行する -> トークン名/トークルームを設定 -> 発行する

発行されたトークンをコピーし以下のようにする
```
self.token = "ここにコピーしたトークン"
```
